# DevOps

> [维基百科](https://zh.wikipedia.org/zh-cn/DevOps)

DevOps 是一种重视“软件开发人员”和“IT 运维技术人员”之间沟通合作的文化、运动或惯例。透过自动化“软件交付”和“架构变更”的流程，来使得构建、测试、发布软件能够更加地快捷、频繁和可靠。 传统的软件组织将开发、IT 运营和质量保障设为各自分离的部门，在这种环境下如何采用新的开发方法，是一个重要的课题。
